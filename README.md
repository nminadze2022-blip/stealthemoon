## Moonstone Landing Page

A minimalistic, monochrome landing page for the Moonstone non‑profit organization, inspired by the colors and atmosphere of the moon (black, grey, and white).

### Structure

- `index.html` – Main landing page markup
- `styles.css` – Visual design, layout, and moon animation styling
- `script.js` – Small enhancements (floating moon parallax and dynamic year)

### Previewing the page

Open `index.html` directly in your browser, or use any simple static server:

```bash
cd projectstealthemoon
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
